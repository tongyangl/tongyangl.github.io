八大排序java实现

作者:Tong Yang    发布于: 2017-08-05

总结整理下八大排序的算法，用java实现。毕竟都是一些面试，平时使用都要用到的算法

  • 冒泡排序

    1. 冒泡排序：一趟一趟的比较，比较相邻元素的大小，每趟有一个元素沉入最后面的位置
    2. 属于交换类排序
    3. 稳定的排序算法
    4. 平均时间复杂度O(n*n)
    5. 最坏时间复杂度O(n*n)
    6. 空间复杂度O(1)
    7. n较小时适用

    ​

  • 选择排序
    1. ​ *直接选择排序：第i轮选择剩余最小的元素放入到第i的位置上
    2. ​ * 属于选择类排序
    3. ​ * 不稳定的排序算法 ：如58529第一轮选择交换5和2，破换了原来两个5的顺序
    4. ​ 平均时间复杂度O(nn)
    5. ​ 最坏时间复杂度O(nn)
    6. ​ * 空间复杂度O(1)
    7. ​ * 当n较小时适用
  • 插入排序
    1. ​ *直接插入排序：每一轮第i个元素跟前面的i-1个元素从后向前比较，一直比较到合适插入的位置
    2. ​ * 属于插入类排序
    3. ​ * 稳定的排序算法
    4. ​ 平均时间复杂度O(nn)
    5. ​ 最坏时间复杂度O(nn)
    6. ​ * 空间复杂度O(1)
    7. ​ * 大部分有序时适用
  • 快速排序
    1. ​ 通过递归和分治的思想，先确定一个值一般为第一个，然后从右向前遍历一旦后面的小，则第一个和这个数字交换位置，然后从左向右遍历。直到两个指针指向同一个位置。
    2. ​ 平均时间复杂度O(nlog2n)
    3. ​ 最坏时间复杂度O(nn)
    4. ​ * 空间复杂度O(1)
    5. ​ * n较大 时适用

可以这么理解 两个指针 j k分别从左向右和从右向左遍历，

从左向右的一旦发现比key大的就给右边

从右向左的一旦发现比key小的就放左边

这样一直遍历到指针指向同一个位置，这个位置就是jk相同并且是key应该存在的位置

  • 希尔排序
    1. ​ *希尔排序：分组交换的思想，先分组，后交换排序
    2. ​ * 属于交换类排序
    3. ​ * 不稳定的排序算法 ：分组之后交换打破了稳定性
    4. ​ 平均时间复杂度O(nlog2n)
    5. ​ * 最坏时间复杂度O(n^s) 1<s<2
    6. ​ * 空间复杂度O(1)
    7. ​ * n较大时适用

  • 堆排序

    ​

二叉堆是完全二叉树或者是近似完全二叉树。

二叉堆满足二个特性：

1．父结点的键值总是大于或等于（小于或等于）任何一个子节点的键值。

2．每个结点的左子树和右子树都是一个二叉堆（都是最大堆或最小堆）。

当父结点的键值总是大于或等于任何一个子节点的键值时为最大堆。当父结点的键值总是小于或等于任何一个子节点的键值时为最小堆
